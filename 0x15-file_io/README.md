#PROJECT 0x15.c - FILE INPUT/OUTPUT
***
```
The project attempts to:
(a) write a function that reads a text file and prints it to the POSIX standard output.
(b) Create a function that creates a file.
(c) write a function that appends text at the end of a file.
(d) write a program that copies the content of a file to another file.
(e) write a program that displays the information contained in the ELF header at the start of an ELF file.
```
